# 🚀 Quick Setup Guide - Background Link Detection

## What Changed?
Your Axiom Security app now detects phishing links **even when the app is closed**! 

## 📱 2-Minute Setup

### Step 1: Install & Open Axiom
- Install the APK on your device
- Open the app

### Step 2: Grant Permissions
The app will automatically ask for:
- ✅ Camera (for intruder detection)
- ✅ Location (for emergency alerts)
- ✅ SMS (for emergency contacts)
- ✅ Notifications (for alerts)
- ✅ Microphone (for voice alerts)

**Tap "Allow" for all**

### Step 3: Enable Background Protection ⚠️ **MOST IMPORTANT**
A dialog will appear asking to enable **"Background Protection"**

1. Tap **"Enable Now"**
2. Find **"Axiom"** in the list
3. Toggle it **ON** ✅
4. Go back to the app

**This allows Axiom to scan messages even when closed!**

### Step 4: Set Trusted Contact (Optional)
- Go to Settings
- Enter your family member's phone number
- Tap "Save Contact"

## ✅ How to Test It Works

### Test 1: Send a Test Link
1. Open WhatsApp (or any messaging app)
2. Send yourself this message: `http://bit.ly/test-phishing`
3. **Listen for voice alert!** 🔊
4. A **red full-screen alert** should appear

### Test 2: Check the Service
- Pull down your notification bar
- You should see: **"Axiom Security Active - Monitoring notifications for threats"**
- If you see this → ✅ It's working in background!

- Source app name

- Warning details

## 🔋 Battery Impact
- **Less than 2-3% per day**
- The service only activates when you receive notifications
- No constant background processing

---

## ⚠️ IMPORTANT: If Alerts Stop Working

If you stop receiving alerts after some time:

1. **Disable Battery Optimization:**
   - Settings → Apps → Axiom → Battery → Unrestricted

2. **Re-enable Notification Access:**
   - Open Axiom → Settings → "Enable Background Protection"

3. **Reboot Device:**
   - The service auto-starts on boot
